## **How to start website**
### **In command prompt**
1. git clone https://github.com/Berubell9/E-Portfolio-Intern.git
2. cd frontend
3. code .
### **In VSCode**
4. new terminal
5. npm start
6. URL : http://localhost:3000

1. เพิ่มข้อมูล เเก้ไขให้เรียบร้อย

4. firebase deploy
