## **How to start website**
1. open XAMPP
2. start Apache and MySQL
3. เอาไฟล์ .sql เข้า MySQL
### **backend**
4. new terminal
5. node app.js
6. รอจนขึ้น port 3333
### **frontend**
7. new terminal
8. npm start
9. URL : http://localhost:3000
